# Serendipity for Figma

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

See `INSTALL.md` for manual variable import.

Available files: `serendipity-midnight-palette.json`, `serendipity-morning-palette.json`, `serendipity-sunset-palette.json`, `INSTALL.md`.
